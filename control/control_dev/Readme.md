# Carte de développement pour le pilotage des moteurs

## Configuration

Vous devez suivre la procédure de configuration d'EAGLE décrite dans le README situé dans utils/Eagle 

Pour ajouter le projet dans Eagle :

- Aller dans le `Control Panel -> Directories`

  ![ControlPanel](http://fsdportal.com/wp-content/uploads/2012/09/Eagle-control-panel-options.jpg)

- Dans le cinquième champs `Projects` ajouter le répertoire correspondant au répertoire parent du répertoire carte



![](http://fsdportal.com/wp-content/uploads/2012/09/Eagle-Directories-624x243.png)
